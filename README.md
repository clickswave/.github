### What we're working on

**[crossfyre_toolchain](https://github.com/clickswave/crossfyre_toolchain)**

Recon and vuln scanning tools that run in your terminal. Find subdomains, scan ports, discover hidden paths, fingerprint what's running, and check it for vulns. All Rust. Each tool works on its own, no account needed. If you want to run this across a bunch of machines there's a hosted platform at [crossfyre.io](https://crossfyre.io), but the tools don't need it.

**silocat**

Encrypted file sharing. Your files get encrypted in your browser before they upload, so all we ever have is ciphertext. Lose the password and even we can't get it back, which is the whole point. It's live at [silo.cat](https://silo.cat) and you can use it right now.

The repo is private until September while we clean some things up. It's going back open after that, same as before.

### On open source

If you can't read the code, you're just trusting us, and you shouldn't have to. The Crossfyre engines and node agent are open, so you can check exactly what's running on your machines before you run it. The hosted control plane is the part we charge for. SiloCat goes back open in September.
